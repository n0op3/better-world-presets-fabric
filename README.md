# Better World Presets for Fabric
This mod was created to allow easy saving and restoring of world generation settings. It also makes it simple to share your world presets with others!

# Features
- Easy saving of world settings
- Restore with one click
- Easily share presets with others
- Simple and lightweight
- Works on all platforms
